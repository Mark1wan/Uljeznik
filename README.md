# ULJEZNIK

A local multiplayer social deduction party game playable entirely from a single HTML file — no installation, no server, no internet required after the page loads.

**"Uljeznik"** is Croatian for *intruder* or *impostor*.

## How to play

1. Open `Uljeznik_V156.html` in any modern browser.
2. Enter the names of all players (minimum 3).
3. Choose how many impostors there are.
4. Pass the phone or device to each player in turn — they tap to reveal their secret hint word.
5. Impostors receive a vague or unrelated hint (or none at all) while the others share a common word.
6. Everyone discusses, asks questions, and votes — can you sniff out the intruder?

## Features

- **Fully offline** — all word lists and hint maps are embedded; no API calls needed.
- **Bilingual** — supports Croatian and English game modes.
- **Single-file** — the entire game, assets, and background image are self-contained in one `.html` file.
- **Configurable** — choose the number of impostors and word category presets.
- **Mobile-first** — designed to be passed around between players on a phone.

## Running the game

Just double-click `Uljeznik_V156.html` or drag it into a browser tab. No build step, no dependencies.
